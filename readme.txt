I originally created this script because I had thousands of e-mails in my mailbox with gold in them. I didn't want to manually click and delete every single mail and because I didn't want to break anything in the rules (and last I checked it wasn't even possible for a GM to assist with this) I went ahead and created a quick Auto Hot Key Script to make up for this.

In order to get the script up and running you'll need to install Auto Hot Key (https://www.autohotkey.com/). Simply click the download button and install as you would any other program. After installing Auto Hot Key, it should automatically associate .ahk files. Simply double click to get the script up and running.

PLEASE READ ALL INFORMATION BELOW BEFORE STARTING
																				 
Double click the .ahk file and it will automatically get the script up and running. You'll then want to login the game go to the Army Hall (since that's where I created this script and the locations should be pretty similar if you're using a 1920x1080 monitor). Once you arrive there you'll then hit control+d and you'll see a prompt come up. Here you'll want to enter how many mails you have in your inbox and press the OK button. The script will then start going going through your mails and clicking on the first slot of the mail inbox. THIS WILL ONLY CLICK THE FIRST SLOT AND ONLY THE FIRST SLOT OF YOUR MAIL, IF YOU HAVE OTHER ITEMS IN YOUR MAILBOX BEWARE SINCE THEY WILL BE DELETED! THIS SCRIPT IS ONLY INTENDED TO GRAB ALL YOUR GOLD. Please also note it's very unlikely that the script will run correctly the first time since when you open your mail the mail window isn't in a static location. You'll probably want to run this script once, watch where your mouse moves, and then move your mail windows to where it's trying to click.  You can also adjust the clicking locations in the script as well.
